## 12. Design Spec CLI Command Specifications

From context-ledger-design-v2.md lines 787-799:

| Command | Specification |
|---------|--------------|
| init | Creates .context-ledger/, config.json, installs post-commit hook (Husky/Lefthook/bare .git/hooks/) |
| serve | Starts MCP server over stdio |
| query <query> | CLI query for debugging. Call queryDecisions or searchDecisions. |
| stats | Summary by source, kind, scope, evidence type, verification status. foldLedger + aggregation. |
| export --format json or csv | Dump ledger. Reads raw LedgerEvent[] via readLedger. |
| validate | Check JSON validity, orphaned targets, lifecycle violations, stale file refs. foldLedger({ strict: false }) + inspect state.warnings[]. |
| validate --propose-repair | Reviewable repair plan to stdout. No file modification. |
| validate --apply-repair | Apply a reviewed repair plan with explicit opt-in. |
| tidy | Compact inbox.jsonl: remove terminal entries (dismissed/expired/ignored) older than 30 days. readInbox -> filter -> rewriteInbox. |
| backfill --max N | Batch backfill capped at N commits per session (default 5). |
| backfill --resume | Resume interrupted backfill session. |
| setup | Interactive wizard -- delegates to src/setup.ts. |

Design spec on validate (line 279):
  validate checks all invariants and reports violations. Does not auto-repair state machine errors.

Design spec on backfill (lines 458, 466):
  Optional cold-start bootstrap. Default cap: 5 commits per session. Supports save/resume by scope area.

## 14. argv Parsing Strategy (Node Built-ins Only)

process.argv[0] = node, process.argv[1] = script path. Effective args start at index 2.

Command patterns:
  init:              argv[2] = init
  serve:             argv[2] = serve
  query <text>:      argv[2] = query, argv.slice(3).join( ) = query string
  stats:             argv[2] = stats
  export:            argv[2] = export, find --format in rest, next element = value
  validate:          argv[2] = validate, argv[3] = --propose-repair or --apply-repair or undefined
  tidy:              argv[2] = tidy
